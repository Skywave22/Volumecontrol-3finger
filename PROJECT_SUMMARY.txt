KEY FEATURES
================================================================================

✓ Quick Settings Volume Tile
  - One-tap volume control from notification shade
  - Cycles through volume levels
  - Shows standard Android volume UI

✓ 3-Finger Swipe Screenshot
  - Detects 3-finger swipe down gesture
  - Captures screen using MediaProjection API
  - Saves to Pictures/Screenshots folder
  - Optional haptic feedback

✓ Floating Overlay UI
  - Always-on volume control widget
  - Volume buttons and slider
  - Draggable interface
  - Enable/disable from settings

✓ Settings & Customization
  - Gesture sensitivity adjustment
  - Vibration feedback toggle
  - Permission management
  - Real-time status display

✓ Background Service
  - Foreground service for continuous operation
  - Battery-aware optimization
  - Proper resource cleanup
  - Android 12+ compatible

================================================================================
REQUIRED PERMISSIONS
================================================================================

Accessibility Service:
  - BIND_ACCESSIBILITY_SERVICE (for gesture detection)
  - Used by: GestureAccessibilityService

Overlay:
  - SYSTEM_ALERT_WINDOW (for floating widget)
  - Used by: FloatingOverlayService

Storage:
  - READ_EXTERNAL_STORAGE (to access gallery)
  - WRITE_EXTERNAL_STORAGE (to save screenshots)
  - Used by: ScreenshotManager

Vibration:
  - VIBRATE (for haptic feedback)
  - Used by: GestureAccessibilityService

Background Service:
  - FOREGROUND_SERVICE (Android 8.0+)
  - FOREGROUND_SERVICE_MEDIA_PROJECTION (Android 12.0+)
  - Used by: ScreenshotBackgroundService

Screen Capture:
  - CAPTURE_VIDEO_OUTPUT (for MediaProjection)
  - Used by: ScreenshotManager

================================================================================
MAIN COMPONENTS
================================================================================

SERVICES:

1. VolumeQuickSettingsTile (TileService)
   - Provides Quick Settings tile for volume control
   - Handles volume cycling on tile tap
   - Updates tile state based on current volume

2. GestureAccessibilityService (AccessibilityService)
   - Detects 3-finger swipe down gesture
   - Provides haptic feedback
   - Triggers screenshot service

3. ScreenshotBackgroundService (Service)
   - Runs as foreground service
   - Captures screen using MediaProjection
   - Saves bitmap to gallery
   - Manages resource cleanup

4. FloatingOverlayService (Service)
   - Displays floating volume control widget
   - Handles user interactions
   - Manages overlay lifecycle

ACTIVITIES:

1. MainActivity
   - Main entry point
   - Displays permission status
   - Provides toggles for features
   - Navigates to settings

2. SettingsActivity
   - Gesture sensitivity slider
   - Vibration feedback toggle
   - Gesture detection status
   - About section

VIEWMODEL:

1. MainViewModel
   - Manages app state
   - Handles volume control
   - Manages settings
   - Provides LiveData for UI

UTILITIES:

1. ScreenshotManager
   - Captures screen using MediaProjection
   - Converts Image to Bitmap
   - Saves to gallery with timestamp
   - Manages resource cleanup

2. PreferencesManager
   - Manages SharedPreferences
   - Stores user settings
   - Provides preference access methods

3. PermissionManager
   - Checks runtime permissions
   - Handles special permissions
   - Opens system settings
   - Provides permission status

4. BatteryOptimizer
   - Monitors battery level
   - Detects low battery mode
   - Adjusts service frequency
   - Prevents critical battery drain

================================================================================
KNOWN LIMITATIONS
================================================================================

1. Gesture Detection
   - 3-finger swipe detection relies on Accessibility Service
   - May not work with all apps (some apps consume touch events)
   - Gesture detection is disabled when accessibility service is off

2. Screenshot Capture
   - Requires MediaProjection permission (user must grant)
   - Cannot capture secure content (passwords, payment info)
   - Screenshot quality depends on device resolution

3. Overlay UI
   - Cannot be used while app is in background (Android 10+)
   - May be hidden by system UI elements
   - Some apps may block overlay display

4. Battery Optimization
   - Services may be killed by system in extreme low battery
   - Foreground service uses battery even when idle
   - Battery optimization varies by device manufacturer
